# 🛒 Blinkit Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Power BI Sales Dashboard** developed using the **BlinkIT Grocery Sales Dataset**. The dashboard provides comprehensive business insights into sales performance, customer ratings, outlet performance, product categories, and inventory distribution.

The objective of this project is to demonstrate how Power BI can transform raw retail data into meaningful business intelligence through interactive visualizations, KPIs, and dynamic filtering.

# 🎯 Business Objectives

The dashboard aims to answer important business questions such as:

- What is the total sales generated?
- Which outlet types generate the highest revenue?
- How does outlet establishment year affect sales?
- Which product categories contribute the most sales?
- How do outlet size and location influence business performance?
- What is the average customer rating?
- Which outlets have the highest number of products?
- How is sales distributed across different fat content categories?

---

# 📂 Dataset Information

**Dataset Name**

BlinkIT_Dataset.xlsx

**Number of Records**

- **Rows:** 8,523
- **Columns:** 12

### Features

| Column | Description |
|---------|-------------|
| Item Fat Content | Low Fat / Regular |
| Item Identifier | Unique product ID |
| Item Type | Category of the product |
| Outlet Establishment Year | Year the outlet was established |
| Outlet Identifier | Unique outlet ID |
| Outlet Location Type | Tier 1 / Tier 2 / Tier 3 |
| Outlet Size | Small / Medium / High |
| Outlet Type | Grocery Store / Supermarket Types |
| Item Visibility | Shelf visibility percentage |
| Item Weight | Weight of the product |
| Sales | Sales value |
| Rating | Customer rating |

The dashboard contains multiple business visualizations designed for interactive analysis.

## KPI Cards

The dashboard summarizes the overall business performance using KPI cards.

- 💰 Total Sales
- ⭐ Average Rating

## Interactive Filters (Slicers)

Users can dynamically filter the dashboard using:

- Outlet Size
- Outlet Location Type
- Item Type
- Item Fat Content

These slicers allow users to drill down into specific business segments.

# 📌 Dashboard Insights

Some key business insights generated from the dashboard include:

- Supermarkets contribute significantly higher sales than grocery stores.
- Product categories vary considerably in sales performance, helping identify high-demand products.
- Outlet location tier influences revenue generation.
- Sales can be analyzed across outlet establishment years to understand long-term outlet performance.
- Customer ratings provide an additional quality metric alongside revenue.
